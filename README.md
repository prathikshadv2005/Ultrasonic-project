# Ultrasonic Distance Measurement using HC-SR04 and Raspberry Pi 4

## 📌 Overview
This project measures real-time distance using an HC-SR04 ultrasonic sensor connected to Raspberry Pi 4.

## 🛠 Components Used
- Raspberry Pi 4 Model B
- HC-SR04 Ultrasonic Sensor
- 1kΩ and 2kΩ resistors (Voltage Divider)
- Breadboard & Jumper Wires

## 💡 Working Principle
The ultrasonic sensor sends sound waves and calculates distance based on echo return time.
